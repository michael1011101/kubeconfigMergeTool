# kubeconfigMergeTool


Usage:
```
usage: merge.py [-h] [-D DIRECTORY] [-T TO] [-a ADD]

合并kubeconfig文件,用于多集群访问

optional arguments:
  -h, --help            show this help message and exit
  -D DIRECTORY, -d DIRECTORY, --directory DIRECTORY
                        the directory of kubeconfigs
  -T TO, -t TO, --to TO
                        the merged kubeconfig
  -a ADD, -A ADD, --add ADD
                        the kubeconfig need to be added
```
